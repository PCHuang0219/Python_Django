What is ONIE?

Open Network Install Environment (ONIE) defines an open install environment for modern network switches in a Linux kernel and BusyBox environment. By opening up the hardware to multiple operating system vendors, e.g. Edge-corE, etc., ONIE initially enables the "white box" and "bare metal" network switching ecosystem, instead of locking end users into a vertical supply chain. Consequently, end users can smoothly select any desired network operating system. Over time, ONIE has become the de facto install environment across the network hardware industry.

Five highlight features

1. Combines a boot loader with a modern Linux kernel and BusyBox
2. Provides an environment for installing any NOS
3. Disruptive; liberating users from a captive, pre-installed NOS
4. Help automate large scale data center switch provisioning
5. Enables you to manage your switches like you manage your Linux servers

How to update ONIE

ONIE provides an update image for upgrading itself. Once located, the update image is downloaded and executed, which mechanism proceeds like the network installer discovery and executing phase. Likewise, ONIE also provides firmware updates (refers to like BIOS firmware or CPLD/FPGA code) similar to updating ONIE itself with a firmware update image. For more information, please visit the following page: https://github.com/opencomputeproject/onie
