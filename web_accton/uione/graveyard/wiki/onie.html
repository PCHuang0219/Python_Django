Open Network Install Environment

From Wikipedia, the free encyclopedia

Jump to navigation Jump to search Open network install environment ( ONIE ) is an open source "install environment", that acts as an enhanced boot loader utilizing facilities in a Linux/BusyBox environment. This small Linux operating system allows end-users and channel partners to install a network operating system as part of data center provisioning, similar to the way servers are provisioned with an operating system of choice. ONIE enables network switch hardware suppliers, distributors and resellers to manage their operations based on a small number of hardware stock keeping unit (SKUs). This in turn creates economies of scale in manufacturing, distribution, stocking, and return merchandise authorization (RMA) enabling a ecosystem of both network hardware and operating system alternatives.
